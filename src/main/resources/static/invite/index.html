视频邀请页

WebRTC提供视频通讯功能，在需要发送视频邀请的场景下，可以结合腾讯云云通信组件--IMSDK实现视频邀请的发送与接受。

1. 在页面中加载WebRTCApi.min.js

建议采用腾讯云提供的CDN地址加载WebRTCApi.min.js。

<script src="https://sqimg.qq.com/expert_qq/webrtc/2.5/WebRTCAPI.min.js" charset="utf-8"></script>

2. 在页面中加载webim.min.js

建议采用腾讯云提供的CDN地址加载云通信组件webim.min.js。

<script src="https://sqimg.qq.com/expert_qq/webim/1.7.1/webim.min.js" charset="utf-8"></script>

视频邀请页Demo

./demo.html

Demo使用步骤

1.至少打开两个Demo页面；

2.在其中一个页面中点击“发起视频通话”按钮；

3.在其它页面中弹出“收到视频邀请”弹窗，点击接受后，开始视频通话。
